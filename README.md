# ARC Reactor v2.0 (AODW-Next 全模态调研 Skill)

> **⚠️ AI-First Repository**
> 本仓库专为 **AI Agent** (特别是 [OpenClaw](https://openclaw.ai) 生态) 设计。如果你是正在阅读此文档的 Agent，请优先将 `SKILL.md` 作为你的主控协议。

**ARC = Acquire (获取) / Research (科研) / Catalogue (编目)**

面向个人知识体系的“战略级”调研引擎。它不只是简单的网页抓取，而是通过**主从架构 (Orchestrator-Worker)**、**多路反爬组件**与**外网主动查杀**，将一个链接或话题沉淀为一份具备深度、可信且自动融合的实体知识库。

## 🎯 v2.0 终极能力集

- 🕵️ **广域交叉核网 (Active Extended Probing)**：打破信息茧房，自动调用搜索引擎查证原素材真伪，执行[异源打假]并标记 `[VERIFIED]`。
- 🎥 **零成本听写引擎 (Media Extractor)**：内置针对 Apple Silicon (M4 Pro) 优化的本地 AI 听写管线。秒级吞吐 YouTube/Bilibili 乃至抖音视频文稿，零 API 计费。
- 🌩️ **高并发子线派生**：支持在后台静默启动子代理 (Sub-agents) 进行大规模并发采集，不阻塞主聊天，不计上下文损耗。
- 📚 **实体卡片索引**：借鉴 Karpathy LLM Wiki 理念，将信息自动编译入 `knowledge/entities/`，实现“同一主题一活页”。
- 📤 **附件化交付**：严禁聊天框刷屏，所有重型报告、音视频稿件均通过文件附件形式静默投送。

## 📦 环境配给与安装 (Bootstrap)

### 第一步：安装系统级依赖
本项目依赖音频处理与本地 AI 加速库，请在 Mac Terminal 执行：

```bash
# 1. 基础音频处理
brew install ffmpeg yt-dlp

# 2. 安装本地加速版听写引擎 (注意：macOS 需添加 --break-system-packages)
pip3 install --user --break-system-packages mlx-whisper
```

### 第二步：安装 Skill 到 OpenClaw
你可以选择**全区安装**（供所有 Gateway 使用）或**局部安装**（仅供特定 Agent 使用）。

```bash
# 进入你的 OpenClaw skills 挂载区
cd ~/.openclaw/skills/

# 克隆仓库并命名为 arc-reactor
git clone https://github.com/evan-zhang/arc-reactor.git
```

### 第三步：填装“弹药库” (.env 配置)
在你的 `.env` 文件中配置以下环境变量，确保 ARC Reactor 的“查杀扫描仪”有动力：

```env
SEARCH_PROVIDER=brave  # 推荐使用 brave api 或 tavily
SEARCH_API_KEY=YOUR_KEY_HERE
```

## 🛠️ 关键排障与注意事项 (Issue #1 已修复)

### 1. agent-browser 版本冲突
如果您使用 `agent-browser` 遇到 Chromium 无法启动，请确保其版本 **>= v0.25.0**：
```bash
npm install -g agent-browser@latest
agent-browser install
```

### 2. 抖音 (TikTok) 反爬特别指南
由于字节跳动极强的动态签名 (`a_bogus`) 封锁，目前该平台已无法实现在线直接剥壳。
- **推荐方案**：采用「本地文件直读模式」。先将视频保存至 Mac，直接传给 Agent 物理路径。
- **降维打击**：`components/media-extractor/scripts/extract.py "/路径/到/你的视频.mp4"`。

### 3. 模型下载加速（首次运行必看）
首次听写会下载 3GB 的权重包，国内用户请务必使用镜像：
```bash
export HF_ENDPOINT=https://hf-mirror.com && python3 ...
```
